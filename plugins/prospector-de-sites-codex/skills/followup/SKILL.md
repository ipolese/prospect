# Follow-up

1. Executar primeiro o procedimento de `respostas` quando Gmail estiver disponível.
2. Selecionar apenas leads em `proposta` há pelo menos 3 dias e sem registro anterior de follow-up.
3. Escrever no máximo quatro linhas, sem pressão, preço ou urgência, seguindo `proposta-email`.
4. Não incluir URL localhost em mensagem destinada ao cliente. Exigir URL pública acessível.
5. Criar rascunho, não enviar, salvo autorização explícita. Registrar o follow-up somente depois que o rascunho existir.
6. Nunca criar segundo follow-up automático para o mesmo lead.
